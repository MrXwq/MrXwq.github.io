一个构造函数

Promise是一个构造函数

一个异步操作一般是无法用return返回结果给调用者的

Promise本身不是异步

但是内部往往都是封装一个异步任务

基本用法

每当new一个Promise实例，就会调用里面的异步操作的代码，如果不想让它立即被执行，只要在外面封装一个函数

封装promise-API

如果第一个then方法指定的回调函数返回的是另一个Promise对象，那么，第二个then方法指定的回调函数就会等待这个新的Promise对象状态发生变化，再被调用

如果第一个then方法指定的回调函数返回的是一个新的Promise实例，那么，会将此返回作为下一个then回调函数的参数

  1. 如果我们不想第一个回调错误之后阻止下面回调的执行，那么需要给.then失败的回调，然后再在失败的回调中添加return

  2. 如果我们想上面的任意一个失败，下面的执行就没意义的话，此时我们想实现一旦有报错，则立即终止所有的Promise的执行，我们只需要在后面添加一个.catch((err) => {

    console.log(err.message)

    })

super

this.必须使用在super后面，因为子类的实例构建是对父类实例加工`，只有super方法才能返回父类实例

子类的super指向父类的prototype

ES6规定，通过super调用父类的方法时，super会绑定子类的this

父类的静态方法也会被子类所继承
