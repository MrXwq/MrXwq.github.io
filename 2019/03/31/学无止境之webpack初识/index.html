webpack4.x

webpack是基于Node进行构建的，所以，webpack的配置文件中，任何合法的Node代码都是支持的

安装

  1. 你需要先初始化目录

  2. 然后再安装

    你新建的项目需要有两个目录，一个dist（打包后的文件会自动放入到dist目录），一个src（你写的代码目录），webpack.config.js（webpack4.x需要的配置文件）

使用

  1. 在webpack.config.js文件中配置

  2. 打包之只要在控制台输入webpack就可以了

可以不需要配置entry和output，默认从src中打包index.js到dist生成main.js

用到的插件

打包好的main.js是托管到了内存中；所以在项目根目录中看不到；但是我们可以认为在项目根目录中，有一个看不见的main.js

热更新

webpack-dev-server: 是否热更新，指定端口，域名，浏览器使用，压缩。。。

内存添加页面

html-webpack-plugin：在内存html中自动追加打包后的文件

配置

如果需要处理非js类型的文件，我们需要手动安装一些合适的第三方loader加载器

css：

npm i style-loader css-loader -D

url地址(图片，字体库):

npm i url-loader file-loader -D

babel：

npm install –save-dev babel-loader @babel/core @babel/cli @babel/preset-env(包含了所有es语法)

npm @babel/plugin-proposal-class-properties -D

vue：

npm install vue-loader vue-template-compiler –save-dev

npm i vue-router -S

向外暴露成员

ES6中：

​ 导入：import 名字(这个名字可以任意取)，{title as title123(要取其他名字，需要用as)} from ‘路径’

​ 暴露：export default{} ——只允许有一个

​ export let title = ‘nice！’ ———–可以有多个

Node.js中：

​ 导入：var 名称 = require(‘模块标识符’)

​ 暴露：modules.exports 或者 exports
