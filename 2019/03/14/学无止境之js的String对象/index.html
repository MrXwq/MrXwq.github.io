学无止境之JS的String对象

创建字符串对象的方法

字符串在底层是以字符数组保存的，所以可以str[0]访问

var stringObject = new String(“hello”) 或者是： var stringObject = “hello”

寻找字符方法

stringObject.charAt(index)//若index=3 则输出”l” index为字符串的索引值。也可以stringObject[index]

stringObjext.charCodeAt(index)//若index=1，则输出101，e的字符编码为101

String.charFromCode()根据字符编码获取字符

字符串操作方法

concat(任意多个参数):连接两个字符串，不影响原字符串，返回一个新字符串。不过一般都用+号连接字符串

slice(参数1，参数2)若只有一个参数，则从这参数(字符串的索引值)开始截取字符串，到结尾，若有两个参数，参数1为开始截取的位置，参数2为结束截取的位置+1。若参数为负数，则将该参数加上字符串的长度返回，两个参数都是这样

substr(参数1，参数2)若只有一个参数，则从这参数(字符串的索引值)开始截取字符串，到结尾，若有两个参数，则参数1为开始截取的位置，参数2为截取字符串的个数。若参数1为负数，则将参数1加上字符串的长度，若参数二为负数，则将负数变为零

substring(参数1，参数2)若只有一个参数，则从这参数(字符串的索引值)开始截取字符串，到结尾，若有两个参数，参数1为开始截取的位置，参数2为结束截取的位置+1。他的所有负数的参数都会变为零。若参数2比参数1小，则会交换参数

这三个参数返回的都是字符串形式

字符串位置方法

indexOf(参数1，参数2)：参数1为字符串的字符，参数2(字符串的索引值)为从哪开始找，返回参数1所在的索引值，从头开始找。如果参数是一个字符串，那么返回第一个字符出现的地方。

lastIndexOf(参数1，参数2):从尾部往前找，参数1为字符串的字符，参数2(字符串的索引值)为从哪开始往前找。返回参数1所在的索引值

trim()函数会删除前后的空格并返回一个副本

trimLeft()和trimRight()会分别删除左右的空格

字符串转换的方法

toLowerCase()将字符串转换为小写字母toLocalLowerCase()

toUpperCase()将字符串转换为大写字母toLocalLowerCase()

字符串匹配方法

var str = “cat,bat,sat,fat”

var pattern = /.at/

var matches = str.match(pattern)实际和正则法则的exec()一样,返回一个匹配的数组

var matches = str.search(pattern)返回找到的字符的索引值，这个二九为0，若pattern = /at/则为1

replace(参数1，参数2)参数1可以为字符串，也可以为正则表达式，第二个参数为字符串，若第一个参数为字符串，则只会用参数2替换一个，而用正则表达式，匹配全局，则可以全部符合的都替换

它的第二个参数也可以是一个函数，函数里有三个参数，第一个为字符串的匹配项，第二个为匹配到的字符所在的索引值，第三个为原始字符串

split(参数1，参数2)找到字符串中等于参数1的值，用逗号替换该匹配项，返回一个数组，若第一个就为匹配项，最后一个也是匹配项，则数组第一个值和第二个值都是空字符串，参数2为数组的大小 如果传入空串，则让每一个字符都成为数组的值

str.localCompare(参数1)：用参数1与str比较，若参数1应该在该字符串的字母该放在str的前面则返回1，若相同不用换，则返回0，若应该放在后面则返回-1.

alert(String.formCharCode(104，101，108，108，111))//输出hello，做的事情和charCodeAt()相反

支持正则的字符串相关方法

split(/[A-z]/)以字母拆,即使不指定全局都全拆

search()搜索指定内容，没有找到则返回-1，搜索到则出现第一次出现的位置

search(/a[bde]c/)

match(/[A-z]/g)提取符合条件的内容出来，返回的是一个数组

replace(参数1，参数2)，参数1被替换的内容，参数2新内容

replace(/a/g,”!!!”)
